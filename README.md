## 프로젝트 소개

> 슬랙의 workspace를 그대로 클론 구현한 프로젝트입니다. NodeJS로 빌드한 서버에 socket.io로 양방향 통신하여 실시간으로 채팅이 가능합니다.

## 프로젝트 실행

1. 클라이언트 실행

```
npm start
```

2. 서버실행

```
cd back
npm run dev
```
